# bilibili-part-timer

Chrome 拓展，统计 B站分P视频时长，支持区间计算与多选勾选。

## 功能

- **区间计算** — 输入起止集数，一键算出总时长
- **选择模式** — 勾选/拖拽多选任意集数，实时统计
- **倍速换算** — 支持倍速调整，结果自动折算
- **日夜适配** — 跟随 B站主题自动切换，实时生效

## 安装

1. 下载 `bilibili-part-timer.zip` 并解压
2. Chrome 打开 `chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的拓展程序**，选择解压后的文件夹

## 使用

打开任意 B站多P视频，页面右侧会出现 🕒 按钮，点击展开面板即可使用。

欢迎反馈 bug。
